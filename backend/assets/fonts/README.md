# Font Dosyaları

Bu klasöre Türkçe karakterleri destekleyen TTF font dosyası eklemeniz gerekiyor.

## Önerilen Font: DejaVu Sans

1. DejaVu Sans fontunu indirin: https://dejavu-fonts.github.io/
2. `DejaVuSans.ttf` dosyasını bu klasöre kopyalayın
3. PDF'lerde Türkçe karakterler artık doğru görünecek

## Alternatif Fontlar

- Arial Unicode MS (Windows'ta genellikle mevcuttur)
- Noto Sans (Google Fonts)
- Open Sans (Google Fonts)

## Font Dosyası Yolu

Font dosyası şu konumda olmalıdır:
`backend/assets/fonts/DejaVuSans.ttf`
